4.6 FUNDAMENTALS OF COMPUTER SYSTEMS

Hardware are the parts of the computer system we can touch and software are the parts of the computer systems we cannot touch.

Hardware NEEDS software to operate it.

  • 4.6.1.2 Classification of Software
    • System Software: System Software is software that is written to control, operate, and maintain a computer.
    • Application Software: Application Software is software that is written to be used as tools by humans.
  • 4.6.1.3 System Software
    • System Software: System Software is software that is written to control, operate, and maintain a computer.
  • 4.6.1.4 Role of an Operating System (OS)
    • Operating System: An OS provides an interface between the human operator and the complexities of operating the hardware.
  • 4.6.2 Classification of Programming Languages
    • Low-Level Languages: A low-level language is a programming language where each line of code is translated into ONE line of machine code.
    • High-Level Languages: A high-level language is a programming language where each line of code is translated into MANY lines of code.
  • 4.6.3 Types of Program Translator
      • Assembler: This is used to translate Assembly Language code into machine code.
      • Compiler: This is used to translate a high-level language into machine code and leave behind an executable file.
      • Interpreter: This is used to translate a high-level language into machine code one line at a time.
  • 4.6.4 Logic Gates
      • Logic gates are the building blocks of a modern computer system.
