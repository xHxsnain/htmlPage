4.7

Hardware are the parts of the computer system we can touch and software are the parts of the computer systems we cannot touch.

Hardware NEEDS software to operate it.

    4.7.1.1 Internal hardware components of a computer:
    Internal Components Understanding the internal components of the processor is fundamental to understanding the Stored Program Concept.
    It is important to know that the internal components work together to make the machine work just like car parts work together to make a car work.
    By itself, a spark plug does nothing. Put it into an engine and it has a purpose as part of the whole process. Similarly, each component in the processor
    is dependent on the other components to function to make the processor work.
    4.7.2.1 The meaning of the stored program concept:
    The Stored Program Concept
    This is as it is written here, and you need to learn and understand this definition:
    "Machine code instructions stored in the main memory are fetched and executed serially by a processor that performs arithmetic and logical operations"
    What has been described here is the Fetch-Decode-Execute Cycle.
    Rational numbers:
    Rational numbers are the set of all numbers that can be expressed as fraction.
    Irrational numbers:
    Irrational numbers are the set of all numbers that cannot be expressed as a fraction.
    Real numbers:
    Real numbers are the set of all numbers that are what we call "real world quantities".
    They include natural numbers, integers, the rational numbers and the irrational numbers.
    Ordinal numbers:
    Ordinal numbers tell the position of an object in a set of objects.
    For example, this can be an index for an array.
    Counting and measurement:
    We use natural numbers for counting. We use real numbers for measurement.
    Fixed Point Binary Numbers:
    Fixed-point binary numbers represent real numbers with a fixed position for the binary point.
    Floating Point Binary Numbers:Floating-point binary numbers are a method of representing real numbers in binary format,
    allowing for a dynamic positioning of the binary point to accommodate a wide range of values and varying levels of precision.
